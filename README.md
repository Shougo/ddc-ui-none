# ddc-ui-none

None UI for ddc.vim

This UI does not display items.

## Required

### denops.vim

https://github.com/vim-denops/denops.vim

### ddc.vim

https://github.com/Shougo/ddc.vim

### none.vim

https://github.com/Shougo/none.vim

## Configuration

```vim
call ddc#custom#patch_global('ui', 'none')
```
